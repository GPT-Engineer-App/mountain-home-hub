# mountain-home-hub

A real estate hub specializing in modern-style mountain homes in the Smokey Mountain/Appalachian region. The site will feature clean, simple navigation to enhance user experience. Backend functionality is not required at this stage.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/mountain-home-hub.git
cd mountain-home-hub
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
